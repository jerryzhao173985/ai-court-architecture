## 🚀 To Activate (3 Steps)

### 1. Get Bot Secret
```
Visit: https://robot.luffa.im
Copy your bot secret
```

### 2. Configure
```bash
# Edit .env file
LUFFA_BOT_SECRET=<your_secret_here>
LUFFA_BOT_ENABLED=true
```

### 3. Launch
```bash
./run_luffa_bot.sh
```

**That's it!** Your bot is live and ready.

## 🎮 How to Play

### In Luffa:
1. Create a group chat
2. Add your VERITAS bot
3. Type: `/start`
4. Follow the trial
5. Participate as a juror

### Commands:
- `/start` - Begin trial
- `/continue` - Next stage
- `/vote guilty` or `/vote not_guilty` - Cast vote
- `/evidence` - View evidence
- `/status` - Check progress
- `/help` - Show commands
