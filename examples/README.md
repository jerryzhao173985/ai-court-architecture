# Examples

Demo scripts showing individual subsystem usage. Run from project root with `cd src &&`.

| Script | Purpose |
|--------|---------|
| `cache_demo.py` | TTL cache operations — set, get, expiration, stats |
| `metrics_demo.py` | MetricsCollector — agent tracking, session stats, summary output |
| `polling_loop_demo.py` | Luffa Bot API polling loop with message deduplication |

```bash
cd src && python ../examples/metrics_demo.py
```
